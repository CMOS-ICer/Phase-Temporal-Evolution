为第四章所搭建功能模型在i_ToF文件夹中，在安装了MWorsk 2026a的情况下，直接用MWorsk的Syslab打开文件夹中的i_ToF.jl文件，点击运行后，其会直接打开MWorks的Sysplorer调用我们设计好的文中Fig.5所示的i_ToF.mo模型。流程类似Matlab中调用Simulink类似。运行的结果会保存在i_ToF_full_model_results.csv中。文中Table 4的关于iToF数据就来自于此csv文件。另外，第四章Fig.6的数据也通过这个模型验证得出。通过修改i_ToF.jl中的velocities、f、d_start、sim_time等参数，可以直接修改物体运动速度、调制频率、物体运动起始距离、曝光时间等参数。模型所需要配置的参数就在文件的最上方，方便直接修改。
同样的，Do_ToF文件夹内的是为Do_ToF搭建的模型，源自对i_ToF.mo的参数修改。运行完后会输出Do_tof_full_model_results.csv，文中Table 4的关于Do_ToF数据就来自于此csv文件,使用流程也是直接用MWorsk的Syslab打开文件夹中的Do_ToF.jl文件，点击运行后，其会直接打开MWorks的Sysplorer调用我们设计好的Do_ToF.mo模型。通过修改文件中的fa、fb、v_start、v_stop、v_step、sim_time等参数可以直接改变零差频率、外差频率、需要仿真的起始速度、结束速度、速度步进、曝光时间等参数，模型所需要配置的参数就在文件的最上方，方便直接修改。
