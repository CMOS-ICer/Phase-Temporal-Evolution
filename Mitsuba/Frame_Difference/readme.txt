The output of this file is used for the comparative verification of the two-frame direct differential method; upon execution, a folder named "result_XXX" is generated in the same directory. Parameters such as object movement speed, initial object distance, modulation frequency, exposure time, and frame interval can be adjusted by modifying variables within the file (e.g., `SPEED_OBJ1`, `OBJ1_INIT_Z`, `MOD_FREQ`, `EXPOSURE`, `GAP_T`). After execution, the files "depth_map_run1_w512x512_f100MHz_T8.0ms_NT100_spp64_v16.50_v20.00.npy" and "depth_map_run2_w512x512_f100MHz_T8.0ms_NT100_spp64_v16.50_v20.00.npy" in the output folder represent the depth results for two consecutive frames at the specified speed. To proceed, place these .npy files directly into the "8-22" folder located within the "Noise Model Addition and Verification" directory; running "Frame_Diff.m" will then yield velocity calculation results incorporating the noise described in the paper, with the output saved to the "sim_output" folder inside "8-22".
